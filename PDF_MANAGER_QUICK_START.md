## 🚀 How to Use

### **1. Add a Course**
1. Click "Add Course" button (top right)
2. Upload thumbnail (optional)
3. Enter course name
4. Set priority number (lower = appears first)
5. Click "Add Course"

### **2. Add a Chapter**
1. Find your course in the list
2. Click the "Add Chapter" button on the course row
3. Upload thumbnail (optional)
4. Enter chapter name
5. Set priority
6. Click "Add Chapter"

### **3. Add a PDF**
1. Expand the course (click arrow)
2. Find your chapter
3. Click "Add PDF" button on the chapter row
4. Upload cover image (optional)
5. Enter PDF title
6. Paste Google Drive URL (must be shared as "Anyone with the link")
7. Set priority
8. Click "Add PDF"

### **4. Edit Any Item**
- Click the blue pencil icon (✏️)
- Update the fields
- Click "Update"

### **5. Delete Any Item**
- Click the red trash icon (🗑️)
- Confirm deletion
- **Warning**: Deleting a course removes all chapters and PDFs inside it!

### **6. View a PDF**
- Click the green eye icon (👁️) on any PDF
- PDF viewer opens in new page

## ✅ Best Practices

1. **Use Priorities Wisely**
   - Start with 1, 2, 3, 4...
   - Leave gaps (10, 20, 30...) for future insertions

2. **Thumbnails**
   - Use clear, high-quality images
   - Recommended size: 400x300 pixels
   - Optional but makes content more attractive

3. **Google Drive URLs**
   - Always set sharing to "Anyone with the link"
   - Test the link in incognito mode before adding
   - Use Drive links, not download links

4. **Naming**
   - Use descriptive names (e.g., "Chapter 1: Motion" not "Ch1")
   - Avoid special characters in titles
   - Keep names concise for mobile view

## 🆘 Common Issues

### **Q: PDF not opening?**
**A**: Check if the Drive link is shared as "Anyone with the link"

### **Q: Image not uploading?**
**A**: Wait a few seconds, the upload happens automatically

### **Q: Items not in order?**
**A**: Check the priority numbers - lower numbers appear first

### **Q: Can't delete a course?**
**A**: Confirm you want to delete - this will remove ALL chapters and PDFs inside

### **Q: Changes not saving?**
**A**: Check your internet connection and try again
